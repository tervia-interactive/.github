# About Tervia Interactive

Tervia Interactive is an independent software and game technology studio focused on building graphics technology, game engines and experimental interactive software.

The organization was created to develop advanced game technology and tools that enable developers to build interactive experiences, games and graphical applications.

Our primary project is the **InteractiveX Engine**, a custom graphics and game engine designed for flexibility, performance and experimentation.

---

# Main Project

## InteractiveX Engine

InteractiveX is a custom graphics and game engine focused on:

• 2D rendering and graphics systems  
• Experimental game engine architecture  
• Performance and low-level programming  
• Modular systems and tools for developers  

The engine is currently under development and evolving rapidly.
